# 구현 하고 싶은 기술
````````````````````````````````````````````````````
1. 회원가입 (주소 API연동, 이메일 인증, ajax통한 아이디 중복검사)
2. 관리자 페이지(인터셉터 적용, 상품관리, 회원관리)
3. 업로드(이미지 등록, 수정, 삭제)
4. 페이징
5. 구매(장바구니, 구매구현)
6. 댓글?구매평?(등록, 수정, 삭제 REST API사용)
````````````````````````````````````````````````````
